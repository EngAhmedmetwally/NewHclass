# **App Name**: Ahmed Metwally 

## Core Features:

- Product Management: Add, edit, and view products (clothing, accessories, shoes) with details like name, category (rental-only, sale-only, both), sale price, rental price, size, branch, and barcode.
- Barcode Generation: Automatically generate sequential numeric barcodes starting from 900000001 or manually enter barcodes for each product.
- Branch Management: Add and edit branch details including name, primary/secondary phone numbers, address, and notes. Branch details appear on printed receipts and invoices.
- Order Management: Create and manage orders (sales and rentals) with unique sequential order numbers starting from 700000001. Track customer info, branch, cashier, seller, products, rental dates, and rental status.
- Multi-Payment Support: Allow multiple payments for rental orders with details including payment method, amount, date/time, cashier, and shift.
- Financial and Shift Management: Manage cashiers, sellers, and shifts. Track shift start/end times, opening/closing balances, and financial data (cash, Vodafone Cash, InstaPay, refunds).
- Reporting: Generate reports for product sales/rentals, seller performance, cashier/shift financials, and branch summaries.

## Style Guidelines:

- Primary color: A sophisticated gold (#D4AF37), evoking luxury and quality.
- Background color: A desaturated beige (#F5F5DC) to complement the gold and give a premium feel.
- Accent color: A muted red (#A52A2A) to highlight key actions.
- Body font: 'Literata', a serif for readability in Arabic and other text, to display the main text.
- Headline font: 'Belleza', a humanist sans-serif to match art and design-oriented brands. Use to display main screen titles.
- Use elegant, minimalist icons to represent product categories, payment methods, and other key functions.
- Maintain a clean, structured layout to facilitate easy navigation and data management.
- Subtle animations, like fade-in effects or sliding transitions, to enhance user experience.
- Auto responsive mobile and tablet PWA
